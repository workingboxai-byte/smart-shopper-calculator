# 🛒 Універсальний калькулятор "помічник покупця" (Smart Shopper)

Простий, адаптивний та ефективний веб-додаток для швидкого перерахунку вартості товарів у супермаркеті. Допомагає розібратися з хитрощами маркетологів, коли ціна вказана за нетипову вагу чи об'єм.

## 🚀 Основні можливості
* **Мультимовність:** Повна підтримка української та англійської (US) мов інтерфейсу.
* **Універсальність мір:** Розрахунок вартості для ваги (кг/гр), об'єму (л/мл) та довжини (м/см).
* **Різні валюти:** Робота з гривнями (грн), доларами ($) та євро (€).
* **Розумне порівняння:** Можливість зберегти до 3 результатів одночасно. Додаток автоматично знайде найвигідніший товар за чистою ціною за одиницю та підсвітить його міткою **"Вигідно!"**.
* **Адаптивний інтерфейс:** Верстка ідеально підлаштовується під будь-які екрани мобільних телефонів у Chrome та WebView.

## 🛠 Технологічний стек
* HTML5
* CSS3 (Flexbox, адаптивна сітка, мобільні оптимізації)
* Чистий JavaScript (без важких бібліотек для миттєвого відгуку на смартфоні)
